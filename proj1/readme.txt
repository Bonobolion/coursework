This program asks for two integers as user input. It then uses two different methods to calculate the product and outputs the result and the time in nanoseconds it took to preform the multiplication.
The first method is adding the higher number to the result the smaller number amount of times.
The second method is called, peasant multiplication this method is much faster for multiplication.

To compile this program simply cd into the directory and use the make command.
Alternately the command run with the makefile is,
	'g++ -std=c++11 project1.cpp'

Run the program with ./a.out
